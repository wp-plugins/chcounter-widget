A simple plugin to create a widget for the chCounter (http://chcounter.org/)

== Description ==

This is a simple plugin which creates a widget for the chCounter (http://chcounter.org/). The plugin expects the counter software to be installed in /chcounter/.

The title of the counter is set via the administration panel. In the current version you can choose to display the following information with the counter

* Total number of visitors
* Online visitors yesterday
* Currently Online visitors
* Online visitors today
* Maximal visitors online per day
* Maximal visitors online at one time
* Total number of page views
* Total number of page views for current page
* Visitors per day
* Link to the chCounter statistics page

Those options are controlled via the widget options page.

The plugin supports internationalization. Currently supported languages are English and German. For translation into other languages, just create the apporpriate MO-File and place it into the chCounter plugin directory.


== Installation ==

To install the plugin to the following steps

1. Unzip the zip-file and upload the content to your Wordpress Plugin directory.
2. Activiate the plugin via the admin plugin page.
3. Go to the widget page and add it to your sidebar. Don't forget to set the title and parameters to be displayed via the widget options page.
